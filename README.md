# Sistema de Detección de Crisis Emocionales (Versión BERT)

Este proyecto implementa un sistema de apoyo para líneas de ayuda psicológica
que estima el riesgo de crisis emocional en llamadas telefónicas usando
Machine Learning.

Esta versión integra un modelo BERT entrenado externamente (por ejemplo,
en Google Colab) y exportado en formato HuggingFace.
